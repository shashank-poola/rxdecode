## 🩺 Rxdecode: Empowering Informed Healthcare Decisions

Rxdecode simplifies understanding your prescription medicines through an easy-to-use web app. Just upload your prescription photo (handwritten or printed), and our smart system automatically extracts medicine names and provides crystal-clear, no-jargon info within seconds.

🚀 Why Rxdecode?
Understanding your medications shouldn't be difficult. We bring:

✅ Medicine Details
✅ Dosage Instructions
✅ Possible Side Effects
✅ Usage Precautions

No medical jargon. Just clear, trustworthy info.

✨ Key Features
Feature	Description
🔍 Prescription Upload	Snap a pic or upload your prescription in seconds.
🧠 OCR-Powered Text Extraction	Built with Tesseract.js, extracts medicine names from handwritten/printed prescriptions.
💊 Real-Time Medicine Info	Instantly fetches accurate details via trusted medical APIs.
🔐 Privacy & Ethics Focused	Your data stays yours. We don’t store sensitive info or give medical advice.

🛠️ Built With
Frontend: React.js + Tailwind CSS

Backend: Express.js (Node.js)

OCR Engine: Tesseract.js

API Integration: External Medicine Info APIs

Fonts: Bricolage Grotesque, Space Grotesk


🖼️ How It Works
1️⃣ Take a photo or upload your prescription.
2️⃣ OCR Engine extracts the medicine names.
3️⃣ Medicine API fetches detailed info on each drug.
4️⃣ Display everything in a clean, easy-to-understand way.
